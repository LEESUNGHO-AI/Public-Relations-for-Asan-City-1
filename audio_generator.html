🎙️ 아산시 스마트시티 나레이션 생성기

아래 3가지 방법 중 하나를 선택하여 자연스러운 여성 나레이션을 생성하세요.
생성된 s0.mp3 ~ s11.mp3 파일을 audio/ 폴더에 넣으면 홍보 영상에 즉시 적용됩니다.

🏆 추천 방법 1: SPEECHMA (완전 무료, 최고 품질)

  1. speechma.com/korean 접속
  2. 아래 텍스트를 복사 → 붙여넣기
  3. 음성 선택: Korean - Female (Sun-Hi 또는 유사)
  4. Generate → Download MP3
  5. 파일명을 s0.mp3, s1.mp3, ...로 변경
  6. 워터마크 없음, 상업적 이용 가능, 로그인 불필요

🥈 추천 방법 2: TTS.ai (무료, Kokoro 모델)

  1. tts.ai/tools/free-tts 접속
  2. 모델: Kokoro 선택 → 한국어 여성 음성
  3. 텍스트 입력 → Generate → Download
  4. 워터마크 없음, 상업적 이용 가능, 계정 불필요

🥉 추천 방법 3: Microsoft Edge 브라우저 읽어주기

  1. Microsoft Edge 브라우저에서 이 페이지를 열기
  2. 텍스트 선택 → 우클릭 → "소리 내어 읽기"
  3. Edge의 ko-KR-SunHiNeural 음성이 자동 적용 (최고 품질)
  4. OBS Studio 등으로 시스템 오디오 녹음
